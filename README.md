Unique Gifts is a frontend-only online gifting website built using **pure HTML, CSS, and Vanilla JavaScript**.  
Users can customize photo frames, gifts, and surprise videos, add items to a cart, and place an order using a **simulated online UPI payment flow**.

---

##  Features

- Photo Frames, Gifts & Surprise Videos
- Product customization with image upload
- Add to Cart using localStorage
- Quantity management & price calculation
- Online UPI payment (simulation)
- WhatsApp order message integration
- Responsive and animated UI

---

##  Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript

---

##  Notes

- No backend used
- No payment gateway
- Uses only local images
- Suitable for portfolio & learning
